# Machine Learning Classification Models

This repository contains classification models implemented using Python and Scikit-learn.

Models included:

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)

Datasets used:

- Titanic Dataset
- Iris Dataset

Steps performed:

- Data preprocessing
- Feature selection
- Model training
- Model evaluation
- Accuracy comparison

Tools used:

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
